# 2023CMPB：PIMedSeg:渐进式交互式医学图像分割 
Gong X, Wang L, Miao L, et al. PIMedSeg: Progressive Interactive Medical Image Segmentation[J]. Computer Methods and Programs in Biomedicine, 2023: 107776.[论文地址](https://www.sciencedirect.com/science/article/abs/pii/S016926072300442X)

医学图像中准确的对象分割是医学诊断和其他应用中的关键步骤。尽管对自动分割方法进行了多年的研究，但实现临床可接受的图像质量仍然具有挑战性。交互式分割被视为一种有前途的替代方案；因此，我们提出了一种基于渐进式工作流程的新的交互式分割框架，以减少用户的工作量并提供高质量的结果。首先，我们的方法使用我们提出的磁盘和曲线变换对用户提供的区域点击和边缘涂鸦进行编码。然后，使用基于变压器的模块进行细化，该模块从卷积神经网络（CNN）的输出和额外的输入映射中提取有效特征。通过对各种医学图像进行了大量的实验，包括超声波（US）、计算机断层扫描（CT）和磁共振图像（MRI），已经证明了我们的新方法相对于最先进的替代方法的有效性。我们所提出的框架可以使用最少的交互实现高质量的分割，而无需花费大量的手动分割成本。

# Introduction
We propose a progressive interactive medical image segmentation (PIMedSeg) framework that integrates region click-based and edge scribble-based interactions. Region click-based interactions are user-friendly, while edge scribble-based interactions can provide useful prior information. Therefore, our objective is to devise a progressive workflow. There are two interaction patterns. In the first pattern, users provide region clicks to facilitate rapid segmentation, where positive clicks denote foreground and negative clicks indicate background. However, relying solely on region clicks may lead to segmentation quality bottlenecks as the number of interaction rounds increases. To address this issue, we designed another pattern wherein the user provides both edge scribbles and region clicks. To represent the interactive information, we introduce a disk and curve (DAC) transform that encodes the utilized input in a novel way. Moreover, to extract high-level features with exact semantics, we build a transformer-based module named TFineNet, which fully uses interactive information and enables long-range dependency capture.

*	We propose a progressive workflow to enhance segmentation quality and minimize user effort. We utilize region clicks for rapid segmentation, and employ edge scribbles along with region clicks to update areas that are difficult to segment.
*	We introduce TFineNet, a powerful yet straightforward transformer-based module that generates highly effective feature representations with precise semantics.
*	The proposed model is lightweight and generalizable, aligning with the characteristics of desirable interactive segmentation. Extensive experiments have been conducted on a variety of medical images, including ultrasound (US), computerized tomography (CT) and magnetic resonance images (MRI), to verify the effectiveness of our new approach over state-of-the-art methods.
